# 🔗 OpenAI API 连通性检测工具

一个纯前端的 OpenAI API 连通性检测工具，支持自动获取模型列表、逐个检测连通性、实时日志追踪。

## ✨ 功能特性

- 📋 智能剪贴板读取 — 自动从剪贴板提取 API URL 和 Key
- 📝 手动粘贴配置 — 支持 curl 命令、配置文本等格式
- 🔍 模型列表获取 — 自动过滤非对话模型
- ⚡ 批量连通性检测 — 支持单个/选中/全部检测
- 🚀 并发检测 — 可自定义并发数量
- 📊 实时统计 — 通过/失败/平均延迟
- 📝 实时日志 — 可导出检测日志
- 💾 配置记忆 — 本地存储，下次自动加载

## 🚀 使用方式

直接打开 `index.html` 即可使用，无需安装任何依赖。

或者部署为静态网站：

```bash
# Python
python3 -m http.server 8080

# Node.js
npx serve .
```

## 🔒 安全说明

本工具为**纯前端应用**，所有 API 请求均在你的浏览器中直接发出，服务器仅托管静态文件，**不会记录、不会转发**任何 API Key 或敏感信息。
